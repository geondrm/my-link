# 마이링크 (MyLink) - 사용자 시나리오 (User Scenario)

## 1. 방문자 (Visitor) 시나리오: 마이링크 페이지 방문 과정

### 1.1 페이지 접속 및 프로필 확인
- 방문자는 **크리에이터(소유자)가 제공한 통합 링크(`domain.com/displayname`)에 접속**하기 위해 **SNS 프로필 하단이나 게시글의 링크를 클릭**한다.
- 방문자는 **해당 페이지가 누구의 것인지 파악**하기 위해 **화면 상단의 프로필(`username` 및 Bio)을 확인**한다.

### 1.2 링크 탐색 및 이동
- 방문자는 **자신이 원하는 목적지(블로그, 포트폴리오 등)로 이동**하기 위해 **화면에 나열된 링크 리스트에서 원하는 버튼을 클릭**한다.
- 방문자는 **마이링크 페이지를 벗어나지 않고 영상을 시청**하기 위해 **임베드된 미디어(유튜브 등) 재생 버튼을 클릭**한다.

---

## 2. 소유자 (Owner) 시나리오: 링크 관리 과정

### 2.1 새로운 링크 추가 (Add)
- 소유자는 **새로운 외부 사이트를 방문자에게 공유**하기 위해 **대시보드에서 '새 링크 추가' 버튼을 클릭**한다.
- 소유자는 **연결할 주소를 지정**하기 위해 **URL 입력란에 주소를 붙여넣고, 방문자에게 보일 제목을 작성한 뒤 '저장'을 클릭**한다. 
  *(이때, 구글 파비콘 API가 작동하여 해당 사이트의 아이콘이 자동으로 설정됨)*

### 2.2 기존 링크 수정 (Edit) - 인라인 편집
- 소유자는 **링크의 제목이나 주소의 오타를 빠르고 간편하게 수정**하기 위해 **텍스트 옆에 있는 '✏️(연필)' 아이콘을 클릭하여 인라인 편집 모드로 진입**한다.
- 소유자는 **수정된 내용을 즉시 반영**하기 위해 **텍스트를 고친 후 엔터(Enter) 키를 누르거나 입력란 바깥 영역을 클릭**한다.

### 2.3 링크 삭제 (Delete)
- 소유자는 **더 이상 유효하지 않은 링크를 리스트에서 제거**하기 위해 **해당 링크 항목의 우측에 위치한 '삭제(휴지통)' 버튼을 클릭**한다.
- 소유자는 **실수로 삭제되는 것을 방지**하기 위해 **화면에 나타나는 확인창(팝업 모달)에서 '정말 삭제하시겠습니까?' 메시지를 확인한 후 '최종 삭제' 버튼을 클릭**한다.

## 💡 제안하는 추가 개선점 (AI Suggestion)
1. **URL 메타 타이틀 자동 완성**: 소유자가 'URL'을 붙여넣었을 때, 시스템이 해당 URL의 웹페이지 제목(Title)을 자동으로 읽어와서 링크 제목(표시 이름) 입력란에 힌트로 채워주는 기능을 제안합니다. (수동 입력의 번거로움을 크게 줄일 수 있습니다.)
2. **새 탭에서 열기 (방문자 유지)**: 방문자가 링크를 클릭했을 때, 현재 페이지가 넘어가는 것이 아니라 **새 창(_blank)**으로 열리도록 설정하여, 방문자가 마이링크(프로필) 페이지를 계속 켜두고 다른 링크도 쉽게 탐색할 수 있도록 유도하는 것을 제안합니다.
